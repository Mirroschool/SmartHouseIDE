PHPMyManager v1.11
---
Описание:
Удобный и многофункциональный файловый менеджер написанный на языке PHP. Позволяет управлять файлами и папками расположенными на сервере.
Новое:
+Возможность копировать файлы
Особенности: 
- Удобный Интерфейс, приятный дизайн с использованием JavaScript
- Листинг директорий на сервере
- Определение размера файлов/папок, даты их создания и даты изменения
- Возможность загружать файлы на сервер
- Возможность создавать файлы
- Возможность редактировать файлы
- Возможность переименовывать файлы и папки
- Возможность удалять файлы и папки
- Многофункциональный поиск файлов(по расширению или по диопазону размера),а также поиск папок
- Фильтрация по типу файла
-Возможность менять права доступа
-Возможность перемещать файлы
-Скрипт стал более удобным

Установка:
1.Скопируйте папку со скриптом на сервер
2.Отконфигурируйте файл config.php при помощи любого текстового редактора5. 3.Пройдите по ссылке ваш_сайт/папка_со_скриптом/index.php

ПРИМЕЧАНИЕ:
- Не удаляйте style.css в нём содержатся стили админцентра
